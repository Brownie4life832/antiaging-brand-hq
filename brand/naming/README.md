# Naming

Everything for choosing the brand name (still **OPEN**), built on the David Placek / Lexicon Branding method.

- [`creative-brief.md`](creative-brief.md) — the "Identify" phase for this brand: the Diamond + creative framework (positioning-neutral). The input to the exercise.
- [`name-candidates-2026-06-19.md`](name-candidates-2026-06-19.md) — **100 curated candidates**, organized by positioning lane (A–E), with a 15-name "fight-for-it" shortlist and an appendix of strong-but-taken words. Generated from 3,128 raw candidates, screened + adversarially web-verified.

**The method itself** lives as an invocable skill (so it's repeatable on any future naming need):
- [`/.claude/skills/brand-naming/SKILL.md`](../../.claude/skills/brand-naming/SKILL.md) — operational guide (Identify → Invent → Implement, the Diamond, three-teams, screening rubric, field drills).
- [`/.claude/skills/brand-naming/reference/placek-method.md`](../../.claude/skills/brand-naming/reference/placek-method.md) — full digest: principles, the A–Z sound-symbolism chart, the 15-name case file, the DIY founder playbook.

To regenerate or extend the pool, re-run the `placek-naming-exercise` workflow against an updated `creative-brief.md`.

> When a name is chosen, log it in [`../../decisions/decision-log.md`](../../decisions/decision-log.md) and update [`../brand-identity.md`](../brand-identity.md). Picking a name should be done *with* the positioning lane it implies.
