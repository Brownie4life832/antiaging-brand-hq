# Master Index — Devices & Procedures Taxonomy

**Compiled:** 2026-06-21 · **Status:** working master list + run tracker (open to edits)

The consolidated taxonomy of skin-aging **modalities beyond topical actives**, and the run list for
the per-modality deep-research dossiers. Deliberately broad (includes clinic-only and fringe/wellness
modalities) for completeness, mirroring the ingredient [`../ingredients/_MASTER-INDEX.md`](../ingredients/_MASTER-INDEX.md).
Each row gets a full, citation-backed, adversarially-verified dossier using [`_template.md`](_template.md).

**Hallmark vocabulary** and the **evidence-grade scale (A–F)** are shared with the ingredient base.
