# Trends

Category and cultural trends that affect the brand: skin longevity, "pro-aging" sentiment, skin-barrier/skin-cycling, ingredient fads, social/creator dynamics, sustainability/refillables. One file per trend.

Keep these dated — trends move. Note whether each is a tailwind, a fad to avoid, or a threat.

## Files

- `2026-06-trend-landscape.md` — standing v1 survey of the 12 macro-trends (the baseline).
- `<YYYY-MM-DD>-radar.md` — **Kintsugi Trend Radar**, the daily digest of new longevity/skincare/beauty developments (ingredients, competitors, trends, regulatory) with a top "Flagged" section for anything urgent or directly competitive. Produced by the `kintsugi-trend-radar` scheduled task.
