# Ideas

The brand's idea pipeline. This is how Claude always knows **what we're considering and what we've ruled out** — exactly as intended for this hub.

| File | Purpose |
|---|---|
| [`parking-lot.md`](parking-lot.md) | Raw capture. Dump anything here, unfiltered, anytime. |
| [`considering.md`](considering.md) | Ideas actively on the table (with current thinking). |
| [`not-considering.md`](not-considering.md) | Ideas ruled out — **always with the reason** (so we don't relitigate). |

## Flow

```
parking-lot  →  considering  →  decision  →  (built)  or  →  not-considering
```

When you raise an idea, it lands in the parking lot. When we engage with it, it moves to *considering*. When we commit or kill it, it moves out — committed ideas get logged in [`../decisions/decision-log.md`](../decisions/decision-log.md); killed ideas go to *not-considering* with a reason.

**Claude: keep these current.** Before answering "are we doing X?", check here.
