# Ads

Ad concepts, copy, and creative briefs by channel (Meta, Google, TikTok, etc.). One file per concept or campaign-channel (e.g. `meta-launch-concepts.md`, `google-search-copy.md`).

For each ad: the hook, the angle, the visual brief, the CTA, and the target audience. Pair copy here with creative built in Canva/Figma. Keep the kintsugi reframe (repair > erase) at the center — that's the scroll-stopper.
