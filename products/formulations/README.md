# Formulations

Formula briefs / ingredient decks per product — the document you'd hand a cosmetic chemist or contract manufacturer. One file per product once its concept is committed.

Each should include: target actives + concentrations, base/excipients, pH, texture/sensory goals, claims to support, packaging/stability needs, and any "must-not-include" list. Ground every active in a [`../../research/ingredients/`](../../research/ingredients/) file.
